EMPEZAR CON UN CUADERNO

QUÉ HACEMOS / Programa Empezar con un Cuaderno

A veces, empezar a aprender también empieza con tener lo necesario.

El programa Empezar con un Cuaderno nace de una realidad simple pero profunda: para estudiar, también hacen falta herramientas. Un cuaderno, una mochila, una lapicera, una carpeta o un libro pueden marcar la diferencia entre participar en clase o quedarse atrás.

A través de este programa, reunimos y distribuimos útiles escolares, mochilas, libros y materiales básicos para niños, niñas y adolescentes de zonas vulnerables. Nuestro objetivo es que la falta de recursos no sea un obstáculo para aprender, hacer tareas o sostener la trayectoria escolar.

Las donaciones se entregan en los puntos de apoyo escolar de Cuadernos Llenos y en comunidades donde las necesidades educativas son más urgentes. También organizamos campañas especiales al inicio del ciclo lectivo y durante el año, según las demandas de cada espacio.

Este programa invita a personas, familias, comercios, empresas e instituciones a colaborar con materiales o aportes económicos destinados a la compra de útiles escolares.

Un kit puede acompañar mucho más

  • ✓ Para aprender: Cuadernos, hojas, lapiceras, lápices, reglas, carpetas y materiales para las tareas cotidianas.
  • ✓ Para organizarse: Mochilas, cartucheras y elementos que ayudan a cuidar y llevar los materiales a la escuela.
  • ✓ Para seguir avanzando: Libros, recursos específicos y materiales comprados con aportes económicos según cada necesidad.

Impacto

✓

Más estudiantes con materiales para aprender

Cada kit ayuda a que niños y adolescentes puedan contar con recursos básicos para sus tareas y clases.

✓

Menos barreras para participar

Tener los materiales necesarios facilita la organización, la continuidad y la participación escolar.

✓

Una red que transforma pequeños aportes

Cada cuaderno, mochila o lapicera se convierte en parte de una comunidad que acompaña trayectorias.

Un cuaderno puede parecer pequeño. Para quien lo necesita, puede ser el comienzo de algo grande.
